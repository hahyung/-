연산자
 /	: 
 %	: 나머지
* 나눌 때 둘 다 정수값을  사용하면 몫만 나온다 
* 소숫점까지 표현하고 싶을 땐 double을 사용하여 한 수를 실수로 만들어 계산한다. 

복합대입연산자
a = 10, b = 5;
a = a + b; => a+=b;
a = a * b; => a*=b;

관계연산자
a=10, b=5;
a > b : true
a <= b : false
a == b : false
a != b : true
10 > 5 > 2 : 사용 못 함
관계연산자는 이항연산자이며 연산자 사이 피 연산자가 2개 온다

논리연산자 : 참 또는 거짓 결과
&&(and) : 모두가 참 일때 참
||(or) : 하나라도 참이면 참
!(not) : 반전시켜준다.

증감연산자 : 결과적으로 자기자신을 1증가 또는 감소
a=10;
++a : 전치 또는 전위
a++ : 후치 또는 후위
a++, ++a => a = a+1;
a--, --a => a = a-1;

삼항연산자(조건연산자)
변수 = 식 ? 참(값) : 거짓(값); 
식의 값이 참이면 : 전의 값을 저장 값이 거짓이면 : 다음의 값을 저장

제어문 : 프로그램의 흐름을 제어한다
if, switch, for, while, do while
